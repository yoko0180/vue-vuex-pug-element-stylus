## Usage

This is a project template for [vue-cli](https://github.com/vuejs/vue-cli). 

``` bash
$ npm install -g vue-cli
$ vue init yoko0180/vue-vuex-pug-bootstrap-stylus my-project
$ cd my-project
$ npm install
$ npm run dev
```
